# homebrew-skicli

Homebrew tap for installing `ski`.

## Install

```bash
brew tap Z-Bra0/skicli
brew install skicli
```

This installs the `ski` command:

```bash
ski version
```

## Upgrade

```bash
brew upgrade skicli
```
